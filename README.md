# Retail Sales Analysis

## Project Overview

This project analyzes a retail sales dataset to uncover business insights such as top-performing products, regional sales distribution, customer segments, and seasonal trends.

The analysis was performed using **Python (Pandas, Matplotlib, Seaborn)** and visualized further using **Tableau Dashboard**.

---

## Tools & Technologies

* Python
* Pandas
* Matplotlib
* Seaborn
* Tableau

---

## Dataset

The dataset contains retail transaction information including:

* Order Date
* Product Category
* Sales
* Customer Segment
* Region
* City

File used:

```
superstore.csv
```

---

## Key Analysis Performed

The following analyses were conducted:

* Data Cleaning and Preparation
* Sales by Category Analysis
* Regional Sales Comparison
* Customer Segment Analysis
* Top 10 Products by Sales
* Top Cities by Revenue
* Monthly Sales Trend

---

## Key Insights

* **Technology** category generated the highest revenue.
* **West region** contributed the largest share of sales.
* **Consumer segment** had the highest purchasing volume.
* Sales showed seasonal trends across months.
